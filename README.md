# Step-1: 간단 야구 게임 구현하기
 
## 간단 야구 게임 설명 
간단 야구 게임은 자동으로 진행되는 야구 게임이다. 게임 시작과 동시에 타자가 타석에 들어서며 게임이 진행된다. 투수의 공은 스트라이크, 볼, 아웃, 안타 중 랜덤으로 정해진다. 게임이 끝나면 최종 안타 수가 출력된다. 

## 코드 동작 방법
* 게임의 시작은 `게임 시작!!` 버튼으로 시작된다. 
* `게임 시작!!`버튼이 클릭되면 핸들러 함수인 init()이 실행된다.
    * 유저가 게임 진행 상황을 알 수 있게 setTimeOut 함수를 사용하여 1.5초 간격으로 게임이 진행된다. 
* 게임 시작과 함께 game.pickRandomCondition 메소드로 스트라이크, 볼, 안타, 아웃을 랜덤으로 얻는다. 
    * 여기서 Condition은 스트라이크, 볼, 안타, 아웃을 의미
* 뽑힌 컨디션은 game.updateCondition 메소드를 통해 개수가 누적된다. ex) 2스트라이크 또는 3볼 등
* 만약 스트라이크가 3개면 1아웃으로, 볼이 4개면 1안타로 바꿔준다. game.updateCondition 메소드 
* game.progress 메소드를 통해 진행 상황을 확인하고 결과를 출력한다.  
* 아웃이 3개가 되면 최종 안타 수를 출력하며 게임이 종료된다.
